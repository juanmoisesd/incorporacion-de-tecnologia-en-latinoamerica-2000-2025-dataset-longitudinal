## Overview
Dataset longitudinal (2000-2025) sobre indicadores de adopción de tecnología digital en 21 países de América Latina e Iberia. Variables: penetración de internet, suscripciones móviles, adopción de smartphones, usuarios de redes sociales, e-commerce, pagos digitales, usuarios de IA, índice de habilidades digitales y prevalencia de ansiedad tecnológica. Fuentes: UIT, Banco Mundial, GSMA, Statista.
